# 노드와 연결

## 배열 리스트의 단점

- 배열에서 사용하지 않는 공간은 낭비된다.
- 배열의 중간에 데이터를 추가하기 위해 데이터를 오른쪽으로 이동해야 한다.

## 노드와 연결

- 낭비되는 메모리 없이 딱 필요한 만큼만 메모리르 확보한다.
- 앞이나 중간에 데이터를 추가, 삭제할 떄도 효율적이다.

```java
public class Node {
    Object item;
    Node next;
}
```

- 노드 인스턴스를 생성하고 item에 데이터를 넣는다.
- next에 다음에 만든 노드의 참조값을 넣는다.
- node.next로 다음 노드를 구할 수 있다.
